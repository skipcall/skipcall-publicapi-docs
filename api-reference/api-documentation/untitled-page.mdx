## **Pagination Parameters**

| **Parameter** | **Type** | **Default** | **Description**                    |
| :------------ | :------- | :---------- | :--------------------------------- |
| `page`        | integer  | 0           | Page number (0-indexed)            |
| `size`        | integer  | 10          | Number of results per page (1-100) |
